# Cài đặt Agent

- **Bypass Planning Mode:** Bỏ qua bước lập kế hoạch (Planning Mode). Khi người dùng yêu cầu sửa đổi mã nguồn, thêm tính năng hoặc fix bug, không cần tạo `implementation_plan.md` hay hỏi ý kiến xác nhận (request_feedback) nữa. Hãy tiến hành thực thi và sửa đổi code ngay lập tức.
- Thực thi công việc một cách nhanh chóng và gọn gàng nhất.
- **Auto Push to GitHub:** Từ giờ trở đi, sau khi chỉnh sửa, cập nhật code hoặc hoàn thành một tính năng, HÃY TỰ ĐỘNG chạy các lệnh `git add`, `git commit` và `git push` để đưa code lên GitHub mà không cần hỏi lại người dùng.
